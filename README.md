## Featured Project

### [CredVerify](https://github.com/MrPiyush67/CredVerify-full) — Certificate Verification Platform

> Reducing certificate verification time from days to seconds using AI, OCR, and blockchain.

Built this solo for SIH 2025. The problem was real — fake certificates are rampant in online education, and verification was manual, slow, and unreliable.

**What I built:**
- 8-stage verification pipeline combining OCR (Tesseract, Surya) and LLMs (Google Gemini) — achieving 85–95% text extraction accuracy with 70% auto-approval rate
- Chrome Manifest V3 extension supporting 50+ platforms (Coursera, Udemy, NPTEL, GitHub, LeetCode) with perceptual hashing for tamper detection
- Decentralized storage on Pinata IPFS + Sepolia Ethereum blockchain with Solidity smart contracts for SHA-256 fingerprint-based certificate recording
- Real-time communication via Socket.IO across a 4-server backend architecture
- 35,000+ lines of code · 50+ REST APIs · JWT auth with role-based access control

`React 19` `Node.js` `Express` `MongoDB` `Ethereum` `Solidity` `IPFS` `Socket.IO` `Google Gemini` `Tesseract OCR`
